How it works
1. Citizen pins location → /api/geo/resolve maps GPS to ward + authority slug
2. Photos validated → /api/reports/validate-images runs Gemini Vision quality check
3. AI triage → /api/reports/{id}/analyze classifies type, severity, and routes to authority
4. Duplicate check → /api/reports/{id}/check-duplicate prevents noise, encourages corroboration
5. Submit → /api/reports/{id}/submit creates live issue with SLA deadline
6. Authority acts → /api/authority/issues shows urgency-scored queue; status updated via /status
